Hornet 2 - двухмоторная электрическая лебедка высокой производительности, разработанная компанией Red-Winches специально для соревнований 4х4 и обладающая всеми характеристиками, необходимыми для автомобильного спорта с высокими эксплуатационными требованиями.

Лебедка RED Winch Hornet в действии!

Основные изменения в лебедках нового поколения Hornet 2: 

  • Тормоза устанавливаются сзади моторов, между мотором и  концевым корпусом, при этом устраняется необходимость использования  дополнительной навески с внешней стороны лебедки.
  • Пространство для установки лебедки увеличено по высоте и ширине, поэтому, по желанию, можно использовать электромотор серии BOW3.
  • Увеличен моторный отсек и шина заземления. Значит, на лебедку можно намотать больший по длине трос, не задевая при этом электромоторы.
  • Повысилось качество прокладок для приводного вала.
  • Используется герметичный барабан (опция). Таким образом устраняется вероятность просачивания воды через подшипники барабана.
  • Шестерни стали легче, шире и прочнее, что позволяет проводить дальнейшую модернизацию электромоторов.
  • Повышена тяговая мощность лебедки. 
  • По сравнению с оригинальной моделью, за счет использования доработанного редуктора несколько повысилась скорость лебедки.
  • По сравнению с оригиналом масса лебедки уменьшена на 1.5кг.
  • Установлен дополнительный монтажный кронштейн для установки пневмоклапана сзади, между блоками Albright.
  • Барабан свободного наматывания оснащен надежным, пневматическим рычажным переключателем.
  • Гарантия на механическую часть до настоящего времени составляет 2 года.
  • Все эти доработки распространяются и на модели  “XL" и “S”
